#Savings Goal Tracker
The python program helps its users calculate how long it will take to reach a savings goal based on how much money they save eachweek . It is useful for financial planning.
## How to Run
Make sure python is installed, then run:
pthon app.py
or
python3 app.py
## Example Output
Welcome to my savingss goal tracker!
what is your savings goal amount? 1000
How much do you save per week? 100
It will take approximately 10.0 weeks to reach your savings goal.
